## 动画

#### transform

1. rotate(45deg):2d顺时针旋转45度
2. scale(1.2):放大
3. translate(x,y)：水平，垂直移动



#### transition：0.5s all(所有属性改变)，all也可以换成height和width，只对高度和宽度进行改变



#### animation(和@keyframes结合用)

```html
<style>
	animation:rainbow 0.5s infinate alternate
  
  @keyframes{
    0%{}
    50%{}
    100%{}
  }
</style>
```
