#### 媒体类型：

all：所有

print：适用于打印的东西

```html
<style>
  .main {
            width: 200px;
            height: 200px;
            background-color: aqua;
            font-size: 12px;
            transition: all 0.5s;
        }

@media only print{
            .main{
                font-size: 50px;
            }
</style>
```

Screen：适用于屏幕

```html
<style>
  .main {
            width: 200px;
            height: 200px;
            background-color: aqua;
            font-size: 12px;
            transition: all 0.5s;
        }
  
	@media only screen and (min-width:300px) and (max-width: 500px) {
            .main{
                background-color:black;
                width: 300px;
                height: 300px;
            }
</style>
```



and：连接，多个一起

not：排除某个特性

only：指定特定的



#### 宽和高：

width，height：显示区域的宽度和高度

min-width，max-width

device-width，device-width:设备的宽度
